# Machine
This repository contains scripts and configuration files for setting up and managing the software and containerized applications I use on my Fedora desktop.
## Fedora Setup
The `fedoraSetup.sh` is a script for automating the installation and configuration of Fedora on my machine. This includes setting up the desktop environment, installing software, and configuring system settings.
## Scripts
The `scripts` directory contains various scripts for automating tasks and managing my Fedora desktop.
## podman
The `podman` directory contains configuration files for managing containerized applications on my Fedora desktop. This includes scripts for building and running containers, as well as configuration files for containerized applications.
See the [podman README](podman/README.md)
